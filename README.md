# NVIM config

Based on kickstarter.nvim

## Requirements

### dprint

```bash
# basic install
curl -fsSL https://dprint.dev/install.sh | sh

# adding package to /usr/bin/local
mv ~/.dprint/bin/dprint /usr/local/bin
sudo chown root:root /usr/local/bin/dprint
sudo chmod 755 /usr/local/bin/dprint
```
